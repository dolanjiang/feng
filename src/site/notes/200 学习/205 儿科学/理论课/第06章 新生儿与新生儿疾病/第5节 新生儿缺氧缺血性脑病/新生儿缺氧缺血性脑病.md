# 新生儿缺氧缺血性脑病

疑似减少：100-150
轻度减少：100-149
中度减少：50-99
重度减少：1-49

早发性：72h内，宫内
晚发性：72h后，感染

病因：
1. 感染
2. 窒息
3. 药物
4. 稀释性血小板减少

治疗：
1. 病因治疗
2. 输血小板，
	1. 小于20-30
	2. 小于50，出血症状
