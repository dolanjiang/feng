# 神经心理发育
## 一、神经系统的发育
新生儿神经细胞已与成人相近。

4岁时神经髓鞘形成，在此之前，尤其婴儿期，刺激引起的神经冲动传导速度缓慢，且易于泛化。（高热惊厥）

3~4月前婴儿肌张力较高，凯尔尼格(kering)征可为阳性。2岁以下儿童巴宾斯基征阳性亦可为生理现象。
