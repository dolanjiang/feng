# M｜万能钥匙 The Skeleton Key (2005)
对这个设定初见的话，伏笔反转和剧情发展还是很合适的，基本满意。女主还算可以吧，矮子里拔将军。看了一小半才想起来自己看过hh，第一次看好像忘记记录了。

Believing is seeing.
