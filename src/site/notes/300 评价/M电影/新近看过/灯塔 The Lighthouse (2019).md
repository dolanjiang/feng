# M｜灯塔
集群是流泻欲望的脓疮，孤寂是通往神性的悬梯。

好吧，实际上，导演在访谈中表示《灯塔》可以解读为一个普罗米修斯遇上海神的故事，延伸而来，参考豆友大佬的评论，可以理解为一种抽象为神话性的讽刺，针对争夺阳具代表的权加上谋求阴部代表的利的男性叙事。
