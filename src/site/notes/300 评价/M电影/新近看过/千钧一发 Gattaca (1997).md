# M｜千钧一发 Gattaca (1997)
很幸运，我小时候就在佳片有约看过它。佳片有约中能留在我记忆里的电影不多，它是其中之一。

不幸的是，当时的我也不会想到：后来这些年，男人之间的友情已经不能满足大众心理，只有所谓的搞基、卖腐才能在部分人贫乏的思想中符合他们对男性深厚情谊的概念。

这是一部极具科幻启示的电影，也是很多人心目中的励志佳作。一句"I never saved anything for the swim back."尽显真正的勇者风范。不过吹毛求疵地讲，在我即将25岁的时候再次细品，还是觉得缺了一点对男主努力和积累实力的刻画（虽然已有不少），即使许多千钧一发的片段都展示了男主优秀的身心特质。

男二自杀是因为男主把梦想借给他了，而梦想就在今天实现，另外身体已经真正借给他人。何由不去，由何再留？不过更深的原因也许是定制的期望与注定的未来。（基因与残疾，在影片中仿佛都是一潭死水的注定。）从男二身上我们也可以看出，歧视与分化、对随机与多样的抹杀、定制的期望与注定的未来对两极都有各自的坏处。

当然，再怎么说，富人的苦恼还是很难与穷人相比的。比如《大佛普拉斯》。
