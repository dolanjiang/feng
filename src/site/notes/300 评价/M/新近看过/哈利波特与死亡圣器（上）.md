# M｜哈利波特与死亡圣器（上）
决战前的准备、压抑和铺垫。这部不是第一次看了。

1. 魔法部的剧情不错，紧张也有笑点。
2. 哈利与赫敏在帐篷的舞蹈虽有争议（多少有点中西差异），但增加了人物弧光。
3. 死亡圣器的童话故事的动画很有趣。
4. 至于压抑的部分，更压抑的电影多了去了。
